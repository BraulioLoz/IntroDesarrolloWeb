Sección 1 - b/strong, i/em, s/del

Esta seccion tendrá información sobre:

  1. Diferencias entre las etiquetas
  2. Implicaciones visuales
  3. Implicaciones respecto a los motores de búsqueda como Google

<b> vs. <strong>

Información <b> <strong>
Diferencias entre las etiquetas Resalta texto poniéndolo en "negritas" sin añadir importancia semántica. Pone el texto en "negritas" añadiendo importancia semántica al motor de búsqueda.
Implicaciones visuales El texto se muestra en negritas, pero sin un significado adicional. El texto se muestra en negritas y se considera importante para el contenido.
Implicaciones al motor de búsqueda No le indica al motor (como google) que es información relevante, solo es un cambio visual Le indica al motor de búsqueda que el texto es relevante al tema.
En una página del itam de las carreras, se debería poner en <strong> los nombres de las carreras y en <b> los nombres de los profesores para resaltar realmente las carreras y darle un énfasis VISUAL a los profesores.

<em> vs. <i>

Información <em> <i>
Diferencias entre las etiquetas Sirve para resaltar texto que es importante o requiere una mayor atención. Se utiliza para que se diferencie estéticamente del resto, por ejemplo, títulos.
Implicaciones visuales El texto se muestra en cursivas y se considera importante para el contenido. El texto se muestra en cursivas, pero sin un significado adicional.
Implicaciones al motor de búsqueda Le indica al motor de búsqueda que el texto es relevante al tema. No le indica al motor (como google) que es información relevante, solo es un cambio visual
En una página de noticias se debería poner en <em> las palabras que son importantes para el tema (como definiciones o conceptos clave) y en <i> los nombres de las secciones para diferenciarlas visualmente.

<s> vs. <del>

En una página como Amazon, se puede poner un <s> en el precio para poder mostrar un precio con descuento, mientras que <del> se usaría para un cambio articulo que ya no se vende actualmente en el sitio.
Informacion <s> <del>
Diferencias entre las etiquetas Se usa para texto que ya no es relevante actualmente. Se utiliza para texto que se quiere mostrar como eliminado.
Implicaciones visuales El texto se muestra tachado, indicando que ya no es relevante. El texto se muestra con una línea a través de él, indicando que ha sido eliminado.
Implicaciones al motor de búsqueda No le indica al motor de búsqueda nada en especial, solo es un cambio visual. Le indica al motor de búsqueda que el texto ha sido eliminado explícitamente del documento.

Sección 2 — Elementos de línea y de bloque

Definiciones:

  • Elemento de bloque: inicia en línea nueva y ocupa todo el ancho disponible (aunque no lo llenen) para crear una estructura de bloques en la página.
  • Elemento de línea: no inicia nueva línea sino que, ocupa el ancho necesario y fluye dentro del párrafo (bloque).

Ejemplos de elementos de bloque

  1. <p> — Párrafo
  2. <div> — Contenedor genérico
  3. <section> — Sección
  4. <article> — Contenido independiente
  5. <header> — Encabezado de página/sección
  6. <footer> — Pie de página/sección

Ejemplos de elementos de línea

  1. <span> — Fragmento en línea
  2. <a> — Enlace
  3. <img> — Imagen “reemplazada” (se comporta en línea por defecto)
  4. <strong> — Importancia
  5. <em> — Énfasis
  6. <label> — Etiqueta de formulario

Como bien se ha visto en la clase, podemos usar un elemento como <div> o <p> para poder agrupar varios elementos dentro de éstos, como información sobre nuestros juegos favoritos, donde dentro de cada uno de los párrafos podemos agregar elementos como enlaces, poner palabras en negritas o cursivas, etc.
